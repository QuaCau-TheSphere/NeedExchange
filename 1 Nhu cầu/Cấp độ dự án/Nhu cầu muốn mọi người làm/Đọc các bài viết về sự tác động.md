Loại đối tượng mình cần họ làm:
```dataview
list where contains(nhu-cầu-của-mình-về-đối-tượng,[[]])
```
Tương ứng với thành quả mong muốn:: 

Các hoạt động để đáp ứng nhu cầu này của mình:
```dataview
list without id hoạt-động from outgoing([[]])
```
